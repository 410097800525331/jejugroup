# 제주 통합 결제창 - 디자인 아이디어

## 선택된 디자인 철학: 모던 미니멀리즘 + 제주 감성

### 디자인 무브먼트
**Contemporary Minimalism with Regional Warmth** - 깔끔한 현대 미니멀리즘에 제주의 따뜻한 감성을 더한 스타일

### 핵심 원칙
1. **명확한 정보 계층**: 사용자가 한눈에 예약 정보와 가격을 파악할 수 있도록 설계
2. **신뢰감과 안정성**: 금융 거래의 안정감을 주기 위해 깔끔한 카드 기반 레이아웃
3. **제주의 자연스러움**: 부드러운 곡선, 따뜻한 톤의 색상으로 제주의 감성 표현
4. **직관적 상호작용**: 명확한 CTA, 부드러운 전환 효과로 사용자 경험 최적화

### 색상 철학
- **Primary**: 제주 바다의 깊이를 담은 `#0066CC` (진한 파랑)
- **Secondary**: 제주 모래사장의 따뜻함 `#FF9500` (주황색)
- **Background**: 제주 하늘의 맑음 `#F8FAFB` (밝은 회색)
- **Accent**: 제주 돌의 차분함 `#6B7280` (중간 회색)

### 레이아웃 패러다임
- **상단 헤더**: 로고 및 네비게이션
- **카드 그리드**: 3개 서비스(항공, 숙박, 렌터카)를 동등하게 표현
- **결제 섹션**: 명확한 가격 정보와 결제 버튼
- **비대칭 공간 활용**: 서비스별 정보를 구분하되 통합감 유지

### 시그니처 요소
1. **둥근 모서리 카드**: 부드러운 `border-radius: 12px`로 친근감 표현
2. **제주 아이콘**: 항공기, 집, 자동차 아이콘으로 각 서비스 시각화
3. **그라데이션 액센트**: 버튼에 미묘한 그라데이션으로 깊이감 추가

### 상호작용 철학
- 호버 시 카드가 살짝 올라오는 효과
- 버튼 클릭 시 부드러운 스케일 변환
- 정보 변경 시 페이드 인/아웃 효과

### 애니메이션
- 페이지 로드: 카드들이 아래에서 위로 부드럽게 올라오기 (0.3s)
- 호버 효과: 카드 `transform: translateY(-4px)` + 그림자 강화
- 버튼: 클릭 시 `scale(0.98)` 피드백

### 타이포그래피 시스템
- **Display Font**: 'Noto Sans KR' Bold (700) - 제목 및 금액 표시
- **Body Font**: 'Noto Sans KR' Regular (400) - 본문 텍스트
- **Accent Font**: 'Noto Sans KR' Medium (500) - 라벨 및 강조

## 구현 방향
- React 컴포넌트 기반 구조 (Home.tsx 메인 페이지)
- Tailwind CSS로 스타일링
- 반응형 디자인 (모바일 우선)
- 부드러운 전환 효과로 프리미엄 느낌 강조
