# 제주항공 브랜드 디자인 레퍼런스

## 브랜드 특징
- **대한민국 No.1 LCC** 제주항공
- 항공 여행의 새로운 기준을 만드는 기업
- 합리적 가격과 혁신이 핵심 가치

## 디자인 요소

### 컬러 팔레트
- **주요 컬러**: 파란색(하늘색) - 항공, 자유, 신뢰
- **강조 컬러**: 주황색/빨간색 - 활동성, 에너지
- **배경**: 흰색/밝은 톤 - 깔끔함, 신뢰성

### 로고 및 브랜드
- 로고: "JejuAir" 이탤릭 스타일
- 항공기 이미지: 파란 하늘 배경에 주황/빨간색 강조
- 현대적이고 역동적인 느낌

### 디자인 철학
- 깔끔하고 현대적
- 신뢰감과 안정성 전달
- 사용자 친화적 인터페이스
- 밝고 긍정적인 톤

## 통합 문의 시스템 적용 방향
1. 제주항공의 파란색(#0066CC 또는 유사)을 주요 컬러로 사용
2. 주황색/빨간색을 강조 요소로 활용
3. 깔끔한 카드 기반 레이아웃
4. 항공, 숙박, 렌터카 세 서비스를 명확히 구분
5. 현대적이고 프로페셔널한 느낌 유지
