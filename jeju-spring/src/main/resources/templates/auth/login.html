Migration Notes

현재 연결 상태

  • 뷰 경로는 `/auth/login`, `/pages/auth/login.html` 둘 다 지원한다.
  • 성공 시 일반 사용자는 `/index.html`, 관리자는 `/admin/pages/dashboard.html`로 이동한다.
  • 실패 메시지는 기존 auth API 응답을 그대로 노출한다.
  • 다음 슬라이스는 `signup` 또는 `session/logout` Spring MVC 전환이다.
